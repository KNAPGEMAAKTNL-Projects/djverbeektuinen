# DJ Verbeek Tuinen

Professional website for DJ Verbeek Tuinen - a landscaping company based in Ophemert, Netherlands.

## Tech Stack

- **Framework**: [Astro](https://astro.build/) v5
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) v4
- **Icons**: [Lucide React](https://lucide.dev/)
- **Fonts**: Merriweather (serif), Inter (sans-serif)

## Services

- **Tuinontwerp** - 3D Garden Design
- **Tuinaanleg** - Garden Construction
- **Grondwerk** - Groundwork & Excavation

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
